# Acting Assignment Management

Manage temporary acting assignments and compensation adjustments.

## Run locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:5005](http://localhost:5005).

Build: `pnpm build`. Start production: `pnpm start`.

## Deploy on Vercel

Connect this repository to [Vercel](https://vercel.com) and deploy with default Next.js settings.

**Environment variable (optional):** Set `NEXT_PUBLIC_EXPIRATION_REMINDER_DAYS` in Vercel if you want to change how many days before an assignment’s end date the expiration reminder is shown (default: 7).
